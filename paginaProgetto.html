Questo progetto nasce con l'intenzione di creare un software gestionale per la gestione dell'inventario e delle offerte per i supermercati.
L'obbiettivo principale era quello di poter facilitare il lavoro nel caricare prodotti nuovi, rimuoverne di vecchi, inserire o modificare scontistiche
e gestire eventuali modifiche ai singoli prodotti.
Le Principali Funzionalità

    • Aggiungi Prodotto
    • Rimuovi Prodotto
    • Aggiungi Offerta
    • Rimuovi Offerta
    • Modifica Offerta
    • Ricerca Offerta per Nome Prodotto
    • Ricerca Offerta per Categoria Prodotto
Le Tecnologie Utilizzate nello Sviluppo


    • HTML
    • CSS
    • BootStrap
    • Java
